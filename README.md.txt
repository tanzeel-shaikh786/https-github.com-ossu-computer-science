Student Grade Tracker (Part 1)

📌 Overview

This project is a command-line based Student Grade Tracker built using basic Python concepts like loops, conditionals, lists, and strings.

The program cleans raw student data, analyzes marks, and generates a class performance report.

---

🧠 Concepts Used

* Data cleaning (strip, title, split)
* Type conversion (string to int)
* Lists and loops (for, while)
* Conditional statements (if-else)
* String manipulation
* Basic statistics (sum, average, max, min)

 ✅ Tasks Implemented

Task 1: Data Cleaning

* Cleaned student names (removed spaces, fixed casing)
* Converted roll numbers to integers
* Converted marks string into list of integers
* Checked if names are valid
* Printed formatted student profiles

---

Task 2: Marks Analysis

* Assigned grades based on marks
* Calculated total and average marks
* Found highest and lowest scoring subjects
* Built a user input system to add new subjects
* Handled invalid inputs safely

---

Task 3: Class Summary

* Calculated average marks for each student
* Assigned Pass/Fail status
* Counted total passes and fails
* Found class topper
* Calculated class average

---

 Task 4: String Operations

* Cleaned essay text
* Converted to title case
* Counted occurrences of "python"
* Replaced text with "Python 🐍"
* Split essay into sentences
* Printed numbered sentences
